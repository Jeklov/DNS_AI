# Android-приложение DNS с ИИ для умного подбора техники

*(на основе приложения магазина DNS ритейл)*

## 1. Общее описание проекта

Проект представляет собой Android-приложение, функционально повторяющее базовые возможности интернет-магазина DNS, с интеграцией **нейронных сетей** для интеллектуального поиска, подбора и конфигурирования товаров.

Основная цель проекта — продемонстрировать, как ИИ может:

* упростить поиск товаров,
* помочь пользователю подобрать подходящее устройство под задачу и бюджет,
* автоматизировать сборку ПК с учетом совместимости комплектующих.

На текущий момент приложение использует **ограниченную базу данных**, полученную путем парсинга сайта DNS, и ориентировано на демонстрацию концепции (MVP).

---

## 2. Основные возможности приложения

### Умный поиск товаров

**Описание:**
Интеграция ИИ-ассистента в стандартный поиск магазина.

**Функциональность:**

* Поиск товаров по запросу на естественном языке
  (например: *«телефон для игр в танки»*).
* Анализ запроса пользователя (цель, тип устройства, приоритеты).
* Формирование списка релевантных товаров.
* Возможность перехода от результатов поиска к карточке товара и покупке.

**Преимущества:**

* Пользователю не нужно разбираться в характеристиках.
* Поиск работает по смыслу, а не по точному совпадению слов.

---

### Чат с ИИ-ассистентом

**Описание:**
Встроенный диалоговый интерфейс для консультаций и подбора техники.

**Возможности чата:**

* Ответы на технические вопросы.
* Подбор товаров под конкретные задачи:

  * игры,
  * учеба,
  * работа,
  * бюджетные решения.
* Навигация пользователя к:

  * умному поиску,
  * конфигуратору ПК,
  * списку товаров.

**Режимы работы:**

* Простой диалоговый режим.
* Режим подбора конфигурации ПК.
* Режим умного поиска.

---

### ИИ-конфигуратор ПК

**Описание:**
Интеллектуальный инструмент для автоматической сборки компьютера.

**Функциональность:**

* Подбор комплектующих под:

  * бюджет пользователя,
  * задачи (игры, работа, универсальный ПК).
* Проверка совместимости деталей.
* Автоматическое формирование итоговой конфигурации.
* Расчет общей стоимости.

## 4. Работа с данными

* Источник данных:
  Спарсенные товары с сайта DNS (ограниченный набор).
* Тип данных:

  * телефоны,
  * ноутбуки,
  * комплектующие ПК.
* В проекте не реализована реальная покупка — используется имитация.

---

## 5. Ограничения текущей версии

* Неполная база товаров.
* Отсутствие реального API DNS.
* ИИ работает в демонстрационном режиме.
* Нет авторизации и личного кабинета пользователя.
